搭建gitee图床-记录

下载软件 node.js / git picgo / 配置好git全局信息
软件下载地址：picgo:https://picgo.github.io/PicGo-Doc/zh/guide/#picgo-is-here
git:https://git-scm.com/download/win
node.js:https://nodejs.org/en/download/prebuilt-installer/current

git配置的话可以前往gitee项目中找一下命令。

  • picgo软件点击插件设置下载gitee-uploader. 如果下载gitee 可能会上传不了。（不晓得麻子情况）

  • 然后前往gitee新建仓库，创建私人令牌即可

  • 最后配置picgo信息

    • branch :分支，这里填master
    • token: 填码云私人令牌，下面介绍生成方法
    • path: 图片在仓库中的存储路径，这里我们存储在img

  • 最后就可以愉快得上传图片了

  • 搭配typora使用

完！
